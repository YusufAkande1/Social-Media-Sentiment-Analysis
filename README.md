# Social-Media-Sentiment-Analysis
Analysis of social media sentiment and engagement using Python and Power BI

# Project Overview

This project analyzes social media data to uncover insights about user sentiment and engagement patterns. The analysis focuses on understanding how sentiment influences interactions such as likes and retweets.

# Objectives
Perform data cleaning and preprocessing
Conduct exploratory data analysis (EDA)
Analyze sentiment distribution
Identify trends over time
Build a regression model to predict engagement
Visualize insights using Power BI

# Tools & Technologies
Python (Pandas, Matplotlib, Seaborn)
Scikit-learn (Linear Regression)
Power BI

# Data Cleaning
Removed missing values and duplicates
Converted timestamp to datetime format
Extracted features (Year, Month, Day, Hour)
Cleaned text data for analysis

# Exploratory Data Analysis
Distribution of sentiments (Positive, Negative, Neutral)
Engagement analysis (Likes vs Retweets)
Correlation analysis between numerical features

# Time Series Analysis
Analyzed posting trends over time
Identified peak activity periods
Examined sentiment fluctuations

# Sentiment Analysis (NLP)
Cleaned text data
Used TextBlob to calculate polarity
Classified sentiment into Positive, Negative, Neutral

# Regression Analysis

A linear regression model was built to predict Likes using:

Retweets
Sentiment Polarity

# Key Insight:

Retweets strongly influence Likes
Sentiment has a moderate impact on engagement

# Dashboard (Power BI)

The dashboard includes:

Users by Country
Sentiment by Platform
Posts over time
Sentiment trend analysis
Engagement metrics (Likes & Retweets)
Interactive filters

# Key Insights
Positive sentiment generally leads to higher engagement
Strong correlation exists between Likes and Retweets
User activity varies significantly over time
Sentiment fluctuates based on events or discussions
